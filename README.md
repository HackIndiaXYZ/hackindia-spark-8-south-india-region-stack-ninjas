## 📌 About the Project

**Tamil Nalam** is a comprehensive, culturally-tailored healthcare web application specifically designed for the regional demographic. The project bridges the gap between modern technology and traditional health practices, offering a heavily localized experience in both **English and Tamil**.

It emphasizes health awareness through regional remedies, dynamic dashboards, and symptom checking, all wrapped in a stunning, highly-responsive User Interface geared towards an engaging patient experience.

---

## 🎯 Objectives

Our goal in **HackIndia Spark 8** is to:

- 💡 Build innovative and regionalized health healthcare solutions.
- ⚙️ Provide seamless bilingual support to rural demographics.
- 🚀 Deliver state-of-the-art UI/UX using modern web standards.
- 🤝 Combine traditional Tamil medical wisdom with modern tracking.

## 🌟 Key Features

- 🌓 **Instant Theme Switching:** Beautifully animated dark and light mode system leveraging custom RGB variable tailwind mapping.
- 🔤 **Deep Localization:** Complete application state-managed toggle between **English and Tamil**.
- 🌿 **Nature's Pharmacy:** Dedicated `Home Remedies` feature focusing on traditional Tamil treatments using kitchen ingredients.
- ✨ **Daily Health Tips Drawer:** Dynamic interactive side panel displaying critical daily routines (Hydration, Karpuravalli for Cold, 8-Walk Method, etc).
- 🏥 **Dashboard & Trackers:** Clean, modular UI tracking metrics and recent health logging history.

## ⚙️ Installation & Setup
```bash
# Clone the repository
git clone https://github.com/your-username/hackindia-stack-ninjas.git

# Navigate to frontend folder
cd hackindia-stack-ninjas/frontend

# Install dependencies
npm install

# Run the development server
npm run dev

# The app will be available on http://localhost:5176/ or port automatically assigned
```
